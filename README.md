This repository contains the final project for Advanced Web Programming, focused on designing and implementing a web-based Student Attendance Management System for Algiers University. The system aims to replace traditional manual attendance tracking with a reliable, digital platform for professors, students, and administrators.

Project Overview

Manual attendance is often time-consuming, error-prone, and inefficient for large classes. This system provides real-time attendance tracking, reporting, and analytics, ensuring better management of student participation and absence justifications.

Key Features

Role-based Access:

Students: View attendance, submit absence justifications.

Professors: Create and manage attendance sessions, view attendance summaries.

Administrators: Manage student lists, generate statistics, import/export data.

Attendance Management:

Create/open/close attendance sessions.

Record and update attendance in real-time.

Track participation and behavior.

Reporting and Analytics:

Automated attendance summaries per course/group.

Charts and statistics for administrators.

Data Management:

Secure authentication with role-based access.

CRUD operations for users, courses, groups, and attendance records.

Import/export student lists compatible with Progres Excel format.

Technologies Used

Frontend: HTML, CSS, JavaScript, jQuery (mobile-first, responsive design).

Backend: PHP with authentication, role handling, and error management.

Database: MariaDB/MySQL.

Deliverables

Design: Figma prototype, ER diagram, database schema.

Frontend: Responsive pages for students, professors, and administrators.

Backend: Complete PHP implementation with attendance workflows and reporting.
